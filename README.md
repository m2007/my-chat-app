# my-chat-app
Чат реализован на Node.js (сервер - Express) с использованием Socket.io. Фронт - Vue.js (Vue CLI 3), Sass.

### 1. Клонирование репозитория

```
git clone https://github.com/m2007/my-chat-app.git
cd my-chat-app/
```

### 2. Установка пакетов

Для фронта:
```
cd client
sudo npm i
```
Для бека:
```
cd server
sudo npm i
```

### 3. Запуск.

# В режиме разработки:
в первом терминале
```
cd client
npm run serve
```
во втором терминале
```
cd server
npm start
```
открывать http://localhost:8080

# Построить проект:
```
cd client
sudo npm run build
cd ../server
npm start
```
на локалке запускаетсся на http://localhost:3000
